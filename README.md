## Detail related to Project

This project contains component called video-player. this component contains code for showing chat as video. i am using passing JSON file name in route param and then using that same file name to get chat details and showing that same chat details in video format.

JSON file is located in asset folder.
Player contains button for play, pause and stop video and seek bar as well. Seek bar can be used to jump to any certain point and then can continue from that point if clicked on play button.
